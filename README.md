# Customer Support Operations Dashboard

## Overview

An interactive Looker Studio dashboard designed to monitor customer support operations, SLA performance, agent productivity, and operational KPIs. The dashboard enables leadership teams to make data-driven decisions through real-time reporting and visual analytics.

---

## Features

- Executive KPI Dashboard
- SLA Performance Monitoring
- Ticket Volume Analysis
- Agent Productivity Tracking
- Call Performance Dashboard
- Interactive Date & Team Filters
- Automated Google Sheets Integration
- Real-time Operational Reporting

---

## Tools Used

- Google Looker Studio
- Google Sheets
- Data Visualization
- Spreadsheet Functions
- Dashboard Design
- Operational Analytics
